Variables y Tipos de Datos

Contenedores que almacenan valores. JavaScript usa tipado dinámico.

Declaración

let   nombre = "Ana";
const edad   = 25;
var   activo = true;

Tipos Primitivos

"texto"    // String
42         // Number
true       // Boolean
null       // Null
undefined  // Undefined

02
🔀

Condicionales

Toma decisiones en tiempo de ejecución según condiciones evaluadas.

if / else if / else

if (edad >= 18) {
  console.log("Mayor");
} else if (edad >= 13) {
  console.log("Teen");
} else {
  console.log("Niño");
}

Operador Ternario

const msg = edad >= 18
  ? "Adulto"
  : "Menor";

// switch también es válido:
switch(dia) {
  case "lunes": …
}

03
🔄

Bucles (Loops)

Ejecuta bloques de código repetidamente mientras se cumple una condición.

for clásico

for (let i = 0; i < 5; i++) {
  console.log(i);
}
// 0, 1, 2, 3, 4

while / for…of

let i = 0;
while (i < 5) { i++; }

for (const x of arr) {
  console.log(x);
}

04
⚙️

Funciones

Bloques de código reutilizables que reciben parámetros y retornan valores.

Declaración Clásica

function saludar(nombre) {
  return `Hola, ${nombre}!`;
}
saludar("Ana"); // "Hola, Ana!"

Arrow Function

const saludar = (n) =>
  `Hola, ${n}!`;

// Con múltiples parámetros:
const suma = (a, b) => a + b;

05
📋

Arrays (Arreglos)

Estructuras que almacenan múltiples valores accesibles por índice numérico.

Creación y Acceso

const frutas = ["manzana", "pera"];
frutas[0];       // "manzana"
frutas.length;  // 2

Métodos Clave

frutas.push("uva");
frutas.pop();
frutas.map(f => f.toUpperCase());
frutas.filter(f => f.length > 4);

06
🧩

Objetos

Entidades que agrupan propiedades (clave‑valor) y métodos relacionados.

Creación

const persona = {
  nombre: "Ana",
  edad:   25,
  saludar() {
    return `Soy ${this.nombre}`;
  }
};

Acceso y Desestructuración

persona.nombre;      // "Ana"
persona["edad"];    // 25
persona.saludar();

const { nombre, edad } = persona;

Manipulación del DOM

Accede y modifica el contenido HTML de la página en tiempo real con JavaScript.

Seleccionar

el.textContent    = "Nuevo";
el.style.color    = "red";
el.classList.add("activo");
el.setAttribute("href", "#");

Eventos

Reacciona a las acciones del usuario usando addEventListener y el objeto Event.

addEventListener

btn.addEventListener("click", (e) => {
  console.log(e.target);
});

Tipos de Eventos

"click"      // Mouse
"keydown"    // Teclado
"mouseover"  // Hover
"submit"     // Formulario
"change"     // Input
